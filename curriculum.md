# Callan Method Curriculum Syllabus

このファイルは、英語学習アプリで使用されるデフォルトのカランメソッドカリキュラムの全貌です。
カランメソッドは反射的にフルセンテンス（短縮形必須）で返答するトレーニングを行います。

---

## Stage 1: Beginner (初心者向け)
基本の文型、身の回りの名詞、位置関係（前置詞）などを学びます。

### Lesson 1: Basic Objects & Colors
| ID | 質問 (Question) | ヒント (Target) | 目標回答 (Expected Answer) | イラスト (Visual Cue) |
| :--- | :--- | :--- | :--- | :--- |
| `s1-l1-q1` | Is this a pen? Is this a pen? | No / Pencil | **No, it isn't a pen, but it's a pencil.** | ペン / 鉛筆 |
| `s1-l1-q2` | Is this a table? Is this a table? | No / Chair | **No, it isn't a table, but it's a chair.** | テーブル / 椅子 |
| `s1-l1-q3` | Is the pen long? Is the pen long? | Yes | **Yes, the pen's long.** | 長いペン |
| `s1-l1-q4` | Is this book black? Is this book black? | No / White | **No, this book isn't black, but it's white.** | 黒い本 / 白い本 |
| `s1-l1-q5` | Is the box small? Is the box small? | Yes | **Yes, the box's small.** | 小さい箱 |

## Stage 2: Elementary (初級向け)
現在進行形、所有代名詞などを学びます。

### Lesson 1: Present Continuous Actions
| ID | 質問 (Question) | ヒント (Target) | 目標回答 (Expected Answer) | イラスト (Visual Cue) |
| :--- | :--- | :--- | :--- | :--- |
| `s2-l1-q1` | What's the teacher doing? | Standing | **The teacher's standing.** | 立っている人物 |
| `s2-l1-q2` | Are you speaking English? | Yes | **Yes, I'm speaking English.** | 話している吹き出し |
| `s2-l1-q3` | Is the student writing in a book? | No / Listening to the teacher | **No, the student isn't writing in a book, but he's listening to the teacher.** | 書く / 聞く人物 |
| `s2-l1-q4` | What's the student doing? | Answering a question | **The student's answering a question.** | 挙手・発話する人物 |

## Stage 3: Pre-Intermediate (中級前向け)
比較級、時間の表現、否定比較などを学びます。
